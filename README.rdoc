==My_Bash

Está é a configuração do meu bash, adaptei pegando a configuração do @fnando: https://github.com/fnando/bash

Para utilizar, basta um clone deste repositório para '~/.my_bash' e adicionar a seguinte linha ao seu '~/.bash_profile' ou equivalente:

        source ~/.my_bash/start.sh

Para instalar a configuração do Pry + Irb modificado execute o script de instalação:

        cd .my_bash/dev
        ./install.sh
